# 📋 订单管理后台使用说明

## 🎉 已上线！

**管理后台地址**：https://qqdwzl.github.io/ordering-app/admin.html

**点餐页面**：https://qqdwzl.github.io/ordering-app/

---

## ✨ 功能列表

### 📊 数据统计
- **总订单数** - 所有订单数量
- **总营业额** - 累计收入
- **待处理** - 待支付订单数
- **今日订单** - 今天的订单数

### 🔍 订单筛选
- **按状态筛选** - 待支付/已支付/已完成
- **按日期筛选** - 选择具体日期查看
- **搜索功能** - 搜订单号或菜品名

### 📝 订单管理
- **查看详情** - 查看订单完整信息
- **标记完成** - 将订单状态改为已完成
- **取消订单** - 取消待支付订单
- **导出 Excel** - 导出 CSV 格式订单数据

### 🔄 自动刷新
- 每 30 秒自动刷新订单
- 手动刷新按钮

---

## 📱 使用场景

### 后厨/店员
1. 打开管理后台
2. 查看新订单（待处理）
3. 制作完成后点击"完成"
4. 导出每日订单报表

### 店主/经理
1. 查看营业额统计
2. 筛选特定日期订单
3. 导出 Excel 做财务分析
4. 搜索特定菜品销量

---

## 🔐 安全提示

⚠️ **当前版本订单数据存储在浏览器本地**

- 订单只在顾客下单的浏览器保存
- 管理后台需要**在同一台设备**上查看
- 清除浏览器数据会丢失订单

### 升级方案（推荐）

接入飞书 API 后：
- 订单实时写入飞书表格
- 任何设备都能查看管理
- 数据永久保存

---

## 💡 使用技巧

1. **快速找订单** - 用搜索框输入订单号后四位
2. **查看今日收入** - 看"今日订单"和"总营业额"
3. **导出报表** - 每天打烊前导出 CSV
4. **批量操作** - 筛选"待支付"后批量标记完成
