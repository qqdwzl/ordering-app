## 🔧 常见问题

### 页面加载失败
- 等待 1-2 分钟（Vercel 首次构建）
- 刷新页面
- 检查 Vercel 控制台是否有错误

### API 错误
- 检查飞书 API 凭证是否正确
- 查看 Vercel 函数日志（Dashboard → Functions）

### 订单未写入飞书
- 检查飞书表格权限
- 查看 Vercel 日志中的错误信息
